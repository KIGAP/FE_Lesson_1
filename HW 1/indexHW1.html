Что такое CMS

               CMS  "система   управления   контентом" (движок) - 
        написанная  PHP-программистами   основа   для сайта, с 
        помощью    которой    вы   можете    управлять     сайтом 
        (добавлять    контент,   менять   пункты   меню   и   т. п.) 
        не знаю HTML и CSS.

              Однако,  для   того  чтобы  сделать  сайт  с  помощью
        CMS потребуются услуги и программиста, и дизайнера,
        и верстальщика. И капиталовложения.

Какие бывают cms

              Бывают   различные  методы управления  контентом:
        для интернет-магазинов,  для  блогов, для форумов и т.д.

Примеры cms

              Примеры   популярный  CM: Joomla,  WordPress (для
        блогов), PhpBB (для форумов).      
        

            CMS-ки бывают платныые и бесплатные.
